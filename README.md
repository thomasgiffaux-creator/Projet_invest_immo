# Analyse du marché immobilier français

## Identification des opportunités d’investissement locatif et patrimonial

### Présentation du projet

Ce projet vise à analyser le marché immobilier français afin d’identifier les communes présentant le meilleur potentiel pour un investissement immobilier.

L’analyse repose sur le croisement de plusieurs dimensions du marché immobilier :
- prix immobiliers
- loyers
- rendement locatif
- dynamique démographique
- activité du marché

L’objectif est de construire des indicateurs d’aide à la décision permettant d’identifier les territoires les plus attractifs pour investir.Le projet se conclut par la création d’un dashboard interactif Power BI permettant d’explorer le marché immobilier et d’identifier les opportunités d’investissement.

### Objectifs du projet

Les objectifs principaux sont :
- analyser les dynamiques du marché immobilier français
- identifier les territoires présentant les meilleurs rendements locatifs
- identifier les territoires offrant un potentiel de valorisation patrimoniale
- construire des scores d’investissement immobilier
- développer un dashboard interactif pour l’exploration des données

### Deux stratégies d’investissement sont étudiées :

Investissement locatif

Objectif : maximiser la rentabilité locative.

Indicateurs utilisés :
- rendement locatif brut
- niveau des loyers
- liquidité du marché immobilier

Investissement patrimonial

Objectif : privilégier la valorisation du capital à long terme.

Indicateurs utilisés :
- croissance des prix immobiliers
- valorisation du marché immobilier
- dynamique démographique
